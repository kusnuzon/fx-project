## ✨ Welcome!

**CARTRIDGE VAULT** is your digital retro console dashboard.  
Just drop a tool into a folder, push to GitHub, and it magically appears.  
All wrapped in a soft pixel-art dark theme that won’t hurt your eyes.

---

## 🌟 Key Features

- 📂 **Auto Scan** – New folders? New tools? They show up instantly.
- 🎮 **Retro Pixel UI** – Inspired by SNES, Game Boy & GBA.
- 🌐 **One CSS to rule them all** – Global theme for dashboard + every tool.
- 📱 **Responsive & Touch** – Gamepad adapts to portrait or landscape.
- 🔌 **Reusable Modules** – Virtual gamepad works for any emulator.

## 🚀 Get Started

1. Fork or clone the repo.
2. Enable **GitHub Pages** on the `main` branch.
3. Keep it **public** (Auto Scan needs public GitHub API).
4. Visit `https://your-username.github.io/fx-project/` 🎉

## 🖥️ Tech Stack

- **Vanilla HTML, CSS, JavaScript** – No frameworks.
- **GitHub Pages** – Free & instant hosting.
- **Nostalgist.js** – Accurate NES emulation.
- **JSZip** – Extract ROMs from ZIP.
- **GamepadZilla** – Touch virtual gamepad.
