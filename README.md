## ✨ Highlights

- 📂 **Drop a tool → instant listing** – No manual edits.
- 🎮 **Soft pixel‑art dark theme** – Inspired by SNES, Game Boy & GBA.
- 🌐 **One global CSS** – Dashboard and every tool share the same style.
- 📱 **Fully responsive** – NES emulator switches between portrait/landscape.
- 🔌 **Reusable modules** – Virtual gamepad works for any future emulator.

## 🛠️ How to use

1. Fork or clone the repository.
2. Enable **GitHub Pages** on your `main` branch.
3. Make sure the repository is **public** (auto‑scan uses the public API).
4. Visit your own `https://<your-username>.github.io/fx-project/`.

---

## 🖥️ Tech behind the pixels

- **Vanilla HTML, CSS & JavaScript** – No frameworks, no dependencies.
- **Nostalgist.js** – Accurate NES emulation.
- **JSZip** – ROM extraction from ZIP files.
- **GamepadZilla** – Virtual touch gamepad.
- **GitHub Pages** – Free, instant hosting.
